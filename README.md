# Frontend Mentor - FAQ accordion solution

Esta é a minha solução para o [desafio FAQ accordion no Frontend Mentor]. Os desafios do Frontend Mentor ajudam-te a melhorar as tuas competências práticas ao construir projetos realistas.


### O desafio

Os usuários devem ser capazes de:

- Ocultar/Exibir a resposta quando a pergunta é clicada.
- Ver o layout otimizado para a interface do dispositivo (Mobile/Desktop).
